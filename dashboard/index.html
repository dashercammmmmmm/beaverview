Help guide

Using BeaverView

  1. Use the Campus tabs to switch between Corvallis, OSU-Cascades, and Hatfield Marine Science.
  2. Click any colored building on the map to open its room list.
  3. Click a room to load its status, devices, and support tools.
  4. The Actions tab has tool shortcuts (XPanel, ScreenConnect, WattBox, ServiceNow).

All data is mock until production connectors are wired up. See api/.env.example for credential placeholders.
